História dos computadores

(Voltar)

Máquinas Mecânicas (séculos XVII-XIX): Pascal e Leibniz criaram as primeiras máquinas de cálculo mecânico.

Máquina Analítica de Babbage (1837): Diseña la Máquina Analítica, precursora dos computadores programáveis.

Tabuladora de Hollerith (1890): Hollerith utiliza tarjetas perfuradas para processar dados de censura, influenciando no desenvolvimento de sistemas automatizados.

Computadores Eletrônicos (década de 1940): Se construiu o ENIAC, a primeira computadora eletrônica, utilizada para cálculos balísticos.

Circuitos Integrados (década de 1950-1960): Kilby e Noyce introduzem os circuitos integrados, reduzindo o tamanho dos componentes eletrônicos.

Tags básicas html

(Voltar)

O h1 é usado para definir conjuntos de diferentes níveis, onde h1 é o maior e o h6 menor.

A tag p é utilizada para definir um parágrafo de texto. Essencial para estruturar o conteúdo textual em blocos compreensíveis.

mark use para realçar ou destacar partes do texto no conteúdo da página. Geralmente é utilizado para realçar os termos da pesquisa.

Diferença entre listas ordenadas e não ordenadas

(Voltar)

Lista ordenada: É usado para apresentar elementos em uma ordem específica.

Lista não ordenada: É utilizado quando os elementos não necessita de uma ordem específica.
